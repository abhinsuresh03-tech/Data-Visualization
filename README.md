# Laptop Data Visualization

This project explores a dataset of laptops with attributes like brand, CPU, RAM, storage, GPU, and price.  
We use **Pandas, Matplotlib, and Seaborn** to clean the data and create insightful visualizations.

## Visualizations
- 📊 Price distribution
- 🏷️ Average price by brand
- 💾 Storage type vs price
- 🧠 RAM vs price correlation
- 🔥 Feature correlation heatmap
